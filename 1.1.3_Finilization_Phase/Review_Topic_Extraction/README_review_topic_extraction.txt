
1.) First install the python dependencies for the program by running the following command in a python terminal:

pip install -r requirements.txt

2.) Open the review_topic_extraction.ipynb program in your choise of python Integrated Development Environment.

3.) Run the review_topic_extraction.ipynb program and after about 5 minutes the program will ask for a user input 
where the user should input the number of the review they would like to view or 0 to skip this function of the program.

4.) Then after about another 4 minutes the review topic extraction program will should be done.
The program takes around 9 minutes to run on an 8 core CPU with 8 threads.
